# Problem
You are given two strings, A and B. Answer, what is the smallest number of operations you need to  
transform A to B?

Operations are:

1.  Delete one letter from one of strings
2.  Insert one letter into one of strings
3.  Replace one of letters from one of strings with another letter

### Input
T - number of test cases

For each test case:

*   String A
*   String B

Both strings will contain only uppercase characters and they won't be longer than 2000 characters.Â 

There will be 10 test cases in data set.

### Output
For each test case, one line, minimum number of operations.

### Example
>Input:<br/>
1<br/>
FOOD<br/>
MONEY<br/>

>Output:<br/>
4<br/>
